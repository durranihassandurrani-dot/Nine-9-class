# Class 9 Peshawar AI Study Hub Website

The website is already created in this repository.

## Where is the website?

Open this file in your browser:

```text
index.html
```

You can double-click `index.html`, or run a local server with the command below and open the shown address.

## Open it locally

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/index.html
```

## What the website includes

- A professional Class 9 study homepage.
- Peshawar textbook style subject library.
- Book cards for English, Urdu, Islamiyat, Pakistan Studies, Mathematics, Physics, Chemistry, and Biology.
- Notes and chapter-help preview section.
- A demo AI study agent for topic-related questions.
- A roadmap for adding real PDFs, student login, and real AI API integration.

## What still needs your content

5. Whether you want the AI agent connected to a real API.

## Deploy online with GitHub Pages

A GitHub Pages workflow is included at `.github/workflows/pages.yml`.

After uploading this project to your GitHub repository and enabling **Settings → Pages → GitHub Actions**, your live link will look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```

See `GITHUB_PAGES_DEPLOY.md` for step-by-step instructions.
