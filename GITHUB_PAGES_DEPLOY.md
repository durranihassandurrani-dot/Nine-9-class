# Deploy this website to your GitHub account

I cannot log in to your GitHub account from this environment, but the website is ready for GitHub Pages deployment.

## Steps to get your live link

1. Create a new GitHub repository, for example:

   ```text
   class-9-ai-study-hub
   ```

2. Upload all files from this project to that repository.

3. Go to your repository on GitHub.

4. Open:

   ```text
   Settings → Pages
   ```

5. Under **Build and deployment**, choose:

   ```text
   GitHub Actions
   ```

6. Push or upload the files. The included GitHub Actions workflow will deploy the website automatically.

7. Your live link will look like this:

   ```text
   https://YOUR-GITHUB-USERNAME.github.io/class-9-ai-study-hub/
   ```

## Files already prepared for deployment

- `index.html` is the website homepage.
- `.github/workflows/pages.yml` deploys the site to GitHub Pages.
- `preview.png` shows the visual preview.
- `start-website.sh` runs the site locally.
